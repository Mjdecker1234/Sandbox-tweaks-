# Sandbox Tweaks - A Comprehensive Tweaking Mod for Mount & Blade II: Bannerlord

## Description

Sandbox Tweaks is a comprehensive configuration mod for Mount & Blade II: Bannerlord that allows you to customize nearly every aspect of the game to your liking. All options are configurable through the in-game MCM (Mod Configuration Menu), and most tweaks can be enabled or disabled individually.

This mod is inspired by and includes all features from **Kaoses Tweaks** (Nexus Mods #2911), updated for the current version of Bannerlord, with additional support for **WarSail DLC** features.

## Features

### Age Tweaks
- Customize ages for different life stages (infant, child, teenager, adult, old age)
- Set maximum age for characters
- Control when heroes come of age

### Item Tweaks
- Adjust weight multipliers for armor and weapons
- Modify price multipliers for all equipment
- Fine-tune item properties by category

### Battle Rewards Tweaks
- Customize renown gain from battles
- Modify influence rewards
- Adjust morale gain multipliers
- Control gold loss on defeat

### Battle Tweaks
- Set battle size (50-2000 troops)
- Modify battle morale effects
- Adjust hideout troop limits
- Enable battle continuation after player death
- Modify troop experience gain
- Customize siege mechanics

### Campaign & Map Tweaks
- Adjust player party movement speed on the campaign map
- Modify damage multipliers for player and troops
- Control combat AI difficulty

### Clan & Party Tweaks
- Set custom party size limits (up to 10,000)
- Configure companion limits
- Enable unlimited wanderers in taverns
- Adjust AI lord party limits

### Workshop Tweaks
- Set maximum workshop count for player
- Adjust bankruptcy thresholds
- Modify workshop profitability

### Character Development Tweaks
- Configure levels needed per attribute point
- Set focus points gained per level
- Customize character progression

### Pregnancy & Family Tweaks
- Adjust pregnancy duration
- Modify labor mortality chance
- Control stillbirth probability
- Set chance of female offspring
- Configure twin probability

### Crafting Tweaks
- Modify XP multipliers for crafting
- Adjust smelting XP gain
- Control refining XP rewards

### Skill, XP, and Learning Tweaks
- Global XP multiplier for all skills
- Learning rate multipliers
- Skill level cap adjustments (up to 1023)
- Individual skill tweaking

### Naval Combat Tweaks (WarSail DLC)
**⚠️ Note: Settings available, full implementation pending WarSail DLC API documentation**
- Ship speed multipliers
- Naval battle size adjustments
- Ship durability modifications
- Naval skill XP multipliers
- Fleet size limits (up to 100 ships)
- Ship cost multipliers
- Port blockade mechanics tweaks
- Port trade bonus multipliers
- Nord faction troop strength adjustments

*Infrastructure and settings are ready; actual patches will be implemented when WarSail DLC classes are reverse-engineered.*

### Other Features
- Auto-lock equipped items to prevent accidental selling
- Many more quality-of-life improvements

Optional:
- WarSail DLC (for naval combat features)

## Installation

1. Download and install all required dependencies (Harmony, ButterLib, MCM)
2. Download the latest release of Sandbox Tweaks
3. Extract the mod folder to your Bannerlord Modules directory:
   - Default location: `C:\Program Files (x86)\Steam\steamapps\common\Mount & Blade II Bannerlord\Modules`
4. Launch the game and enable "Sandbox Tweaks" in the mod menu
5. Make sure Sandbox Tweaks loads **after** Harmony, ButterLib, and MCM

## Configuration

All settings can be configured through the in-game MCM menu:
1. Start or load a game
2. Press ESC to open the menu
3. Navigate to "Mod Options"
4. Select "Sandbox Tweaks"
5. Adjust settings to your preference
6. Most settings take effect immediately without requiring a restart

## Compatibility

This mod uses Harmony patches to modify game behavior, which means:
- ✅ Compatible with most other mods
- ✅ Safe to add to existing save games
- ✅ Settings are saved per-campaign
- ⚠️ May conflict with other mods that modify the same game mechanics
- ⚠️ Some features require a new game or may not fully apply to existing saves
